## 托管隐私政策和支持页面

### 方法 1: GitHub Pages (推荐)

1. 创建新仓库 `echonotes-docs`
2. 上传 `docs/` 文件夹中的文件
3. 启用 GitHub Pages (Settings → Pages → Source: main branch)
4. 访问: `https://[用户名].github.io/echonotes-docs/`

### 方法 2: Notion 公开页面

1. 在 Notion 创建页面
2. 复制内容
3. 点击 "Share" → "Publish"
4. 获取公开链接
